# ProjectTracker.API

A .NET 8 ASP.NET Core Web API for task and project management, built with Entity Framework Core and SQL Server.

## Features

- RESTful API for projects and tasks
- Full CRUD Operations
- Swagger documentation
- Clean Architecture principles

## Tech Stack

- .NET 8, C# 12
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Swagger/OpenAPI
